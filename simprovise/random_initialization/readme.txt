On Windows, MinGW distribution with g++ 13.2.0 and Boost libraries v1.84.0 downloaded from:

https://nuwen.net/mingw.html

Once installed, run gen_rng_states.bat to create the mt19927_states.npy file, which
are the data to initialize 10,000 simprovise random number streams.

Execution time (2024-04) about 470 cpu seconds
